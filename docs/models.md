# OpenRouter Routing Guide for PROMPT_ARCHITECT

## 1. High-Performance Models (Tier 1)
For critical decision-making, strategic planning, and complex reasoning.
- **claude-opus-4**: Best for deep reasoning and system architecture.
- **claude-sonnet-4-5**: Ideal balance of performance and cost.
- **gpt-4o**: Reliable for general enterprise tasks.
- **deepseek-r1**: Extremely low cost with chain-of-thought (CoT).

## 2. Efficiency Models (Tier 2)
For high-volume, lower complexity tasks like classification and summarization.
- **claude-haiku-4-5**: Fast, reliable, and cost-efficient.
- **gemini-2.0-flash**: Blazing fast with long context support.
- **gpt-4o-mini**: The basic building block for simple agents.

## 3. Specialized Models (Tier 3)
Built for specific domains or functionalities.
- **deepseek-v3**: Elite at code generation and technical analysis.
- **cohere-command-r+**: Optimized for RAG (Retrieval-Augmented Generation).
- **mistral-large**: GDPR/EU compliance focused.

## 4. Selection Criteria
- **Task**: Determine context window and reasoning requirements.
- **Cost**: Compare token pricing against complexity.
- **Latency**: Select models based on the required response time.
